# RecursosDigitalesEDU
Repositorio para la gestión y compartición de recursos educativos digitales alineados con estándares internacionales de metadatos (LOM, Dublin Core), especialmente diseñado para universidades y contextos educativos. Incluye flujos de trabajo, roles diferenciados y mecanismos de búsqueda, evaluación, carga y descarga de materiales.
